# car_prices_dataset
Car prices dataset for Azerbaijan's Car Market. I use it for ML and fun.

#### Definitions

    car_names_count
    This file shows the car name and its count in the main database.
    
    car_names_full
    Main database, includes more than 37000 car data


#### Columns

Each file has the following columns (almost):
    
    car_name
    price
    year
    engine
    mileage
    valyuta - currency
    exchange_rate - it is there to convert AZN to USD
    

Check https://github.com/Sunuba/turboaz to see how I benefited from these datasets
